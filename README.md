# A-AI-Lab

This is a short summary on how to run the programme, in case cryptography isn't a topic you are familiar with. To summarise the part before the user interaction, the code generates prime numbers since these numbers are used in generating the public and private keys which are used to encrypt and decrypt the messages. 

NOTE! The code can only make use of the ASCII characters!

To run the programme, download the raw file and add it to the folder (if you have a separate one) for this course. You may need to use a virtual environment (venv) is the code doesn't work on the non-virtual terminal and also would have to download some of the packagaes (MAYBE--- it depends). 

Generating the 2000-bit primes may take time from anywhere between a couple of seconds to maybe a minute. 

After that, you are asked to enter a message (ASCII characters) after which the code gives you the public key component "e" (this isn't the entirety of the public key but I won't go into the details) which you can just copy from the terminal and paste it. 

Next, you should copy and paste the private key component "d".

After all this, you get the ciphertext (a bunch of numbers) and then the decrypted mesage (the original message you entered at the very beginning). 

Enjoy!  :D
